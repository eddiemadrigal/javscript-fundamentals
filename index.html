What is JavaScript?

  • JavaScript is a scripting or programming language that allows you to implement complex features on web pages
  • With JavaScript, user experiences and interfaces become much more elaborate than if a page simply used HTML on their site

Why JavaScript

JavaScript a scripting language that enables you to create dynamically updating content, control multimedia, animate images, and interact with content located on remote servers. JavaScript has been the premier scripting language for web pages since 1995 and continues to this day to grow in popularity
